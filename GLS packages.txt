CRM — Sync local database (run in this order)
--------------------------------------------------------------------------------

homeschool:mirror-core
  Sync CRM metadata to local mirror: Classes, Teachers, Students, Registrations.
  Run FIRST — other sync commands depend on crm_classes being populated.

  php artisan homeschool:mirror-core
  php artisan homeschool:mirror-core --months=3      # attendance months back (legacy)
  php artisan homeschool:mirror-core --full          # ignore incremental logic

  Feeds: crm_classes, crm_students, crm_registrations (legacy), crm_teachers
  Scheduled: daily 01:00 Africa/Casablanca → logs/homeschool-mirror-core.log

---

crm:sync-attendance
  Sync session-presence (attendance) from CRM bulk API into crm_attendance.
  Uses /bulk/session-presence (supports dateFrom/dateTo — correct endpoint).
  Must run AFTER homeschool:mirror-core (needs crm_classes populated).
  Feeds the Suivi Présences fraud-detection calendar.

  php artisan crm:sync-attendance --all
  php artisan crm:sync-attendance --all --months=2
  php artisan crm:sync-attendance --store=50970 --months=1
  php artisan crm:sync-attendance --store=50970 --store=50995

  Options:
    --all           Sync all stores with a crm_store_id
    --store=ID      Sync one or more specific stores (repeatable)
    --months=2      How many months back to sync (default 2, max 12)
    --delay=500     Delay in ms between page requests (default 500)

  Scheduled: daily 01:15 Africa/Casablanca → logs/crm-sync-attendance.log

---

crm:sync-registrations
  Fetch CRM registrations per store and store in crm_registrations.
  Used by churn scoring and the daily CEO report (new inscriptions count).
  Exponential backoff on 429, lock prevents parallel runs.

  php artisan crm:sync-registrations --all
  php artisan crm:sync-registrations --store=50970 --store=50995
  php artisan crm:sync-registrations --all --delay=800

  Options:
    --all           Sync all stores
    --store=ID      Sync specific store(s) (repeatable)
    --delay=500     Delay in ms between pages (default 500)

  Scheduled: daily 02:30 Africa/Casablanca → logs/crm-sync-registrations.log

---

crm:sync-collections
  Fetch CRM payment-collection (receivables/installments) into crm_collection_rows.
  Feeds the Recouvrement dashboard. Without this the page times out (163 pages live).

  php artisan crm:sync-collections --all
  php artisan crm:sync-collections --store=50970
  php artisan crm:sync-collections --all --delay=600

  Options:
    --all           Sync all stores
    --store=ID      Sync specific store(s)
    --delay=400     Delay in ms between pages (default 400)

  Scheduled: daily 02:00 Africa/Casablanca → logs/crm-sync-collections.log

---

crm:snapshot-payments
  Capture a daily snapshot of ALL payments from CRM into crm_payment_snapshots.
  Used by: CEO daily report (revenue), Statistiques charts, Collections perf table.
  Excludes internal transfers (payment_type_id=4) at query time.

  php artisan crm:snapshot-payments
  php artisan crm:snapshot-payments --store=50970
  php artisan crm:snapshot-payments --store=50970 --store=50996
  php artisan crm:snapshot-payments --date=2026-06-01   # snapshot for specific date

  Options:
    --store=ID      Sync specific store(s) — repeatable (omit for all)
    --date=         Date to snapshot (default: today)

  NOTE: 30s pause between centers to avoid 429 rate limits.
  Scheduled: daily 01:30 Africa/Casablanca → logs/crm-snapshot-payments.log

---

crm:churn-scores
  Compute student churn risk scores from LOCAL DB (no API calls).
  Reads crm_registrations + crm_payment_snapshots.
  Run AFTER crm:sync-registrations.

  php artisan crm:churn-scores --all
  php artisan crm:churn-scores --store=50970 --store=50995

  Options:
    --all           Process all stores
    --store=ID      Process specific store(s)

  Scheduled: daily 03:00 Africa/Casablanca → logs/crm-churn-scores.log

---

crm:daily-report
  Generate the daily CEO CRM report (revenue, inscriptions, top centre).
  Reads from local DB only — no live API calls.
  Run AFTER crm:snapshot-payments.

  php artisan crm:daily-report
  php artisan crm:daily-report --date=2026-06-01

  Scheduled: daily 08:00 Africa/Casablanca → logs/crm-daily-report.log
